常见问题

  
访问码会被别人盗用吗? 它本身就是「凭证」,所以请像对待密码一样对待它。如果你怀疑泄露了,回到本页面重新生成,旧码会自动废掉(限频也能挡住一部分滥用)。

  
Semrush key 存在哪里? 加密(Fernet AES-128)存在服务器的 SQLite 里。明文从不写日志。即使数据库被偷,没有 KEY_ENCRYPTION_SECRET 也解不开。

  
用量怎么算? 每次 GPT 调一次 analyzeHotTopics 就是 1 次。一次调用大约消耗 5,000 Semrush API units(默认上限 50 页 + 200 关键词)。
